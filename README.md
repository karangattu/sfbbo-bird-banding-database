# CCFS Bird Photo Tagger

A web application for tagging and organizing bird banding photos stored in Google Drive with persistent metadata storage in Turso database.

## Quick Start

npm install

### Prerequisites

- Node.js 18+
- Google Drive folder with photos
- Google Service Account with Drive API access
- Turso database account (free at https://turso.tech)

### Run

npm run dev

### Installation

Open http://localhost:3000
